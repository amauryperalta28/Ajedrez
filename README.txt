El ajedrez es un juego entre dos personas, cada una de las cuales dispone de 16 piezas móviles que se colocan sobre un tablero dividido en 64 escaques.1 En su versión de competición2 está considerado como un deporte.

El juego esta desarrollado con la herramienta de Microsoft
XNA STUDIO 4.0, en Visual Studio para facilitar el desarrollo de este en el lenguaje C#.
